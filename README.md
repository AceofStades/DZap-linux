# DZap

Drive Zap: A secure, cross-platform data wiping application that works on Windows, Linux, and Android devices.

SIH PS: 25070


To run backend:
```
cd server
sudo go run .
```

To run frontend open a new terminal:
```
npm run start:frontend
```

Dependencies:
```
util-linux
smartmontools
```

Node Dependencies:
```
npm install cross-env --save-dev
```
